# Faturas_Sinistros
Ler Faturas de Sinistros automaticamente

Atualmente o processo de pagamento de faturas em caso de sinistro é efetuado manualmente existindo necessidade inclusivé de cálculos manuais para posterior inserção no TRON

Este projeto cria uma APP no Streamlit, que permite fazer o upload das faturas e depois é gerada a leitura automática

Neste momento a APP dá resposta a faturas da "Casa de Saúde da Boavista" e o próximo passo será evoluí-la para que leia também as dos "Lusíadas"
